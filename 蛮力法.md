## 蛮力法

### 串匹配

#### BF

伪代码：

![image-20201204094238809](/image-20201204094238809.png)

代码：

~~~c++
int bf(string s,string t){
	int i=0,j=0;
    for(i=0;i+t.size()<j.size();i++){
        if(s[i]==t[j]){
            j++;
            if(j==t.size())
                return i=t.size();
        }
        else 
            j=0;
    }
    return 0;
}
~~~
